# Boilerplate

A starter project for React, Redux & Webpack

###Also Includes###
```
*react-router
*testing with mocha, chai, and sinon
*Example Files
```
###Getting Started###

```
	> git clone git@github.com:terrycoco/boilerplate.git <YourProjectName>
	> cd  <YourProjectName>
	> npm install
	> npm start
	> open a browser to localhost:8080
```
